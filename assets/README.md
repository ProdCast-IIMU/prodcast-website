# ProdCast Website – Assets Folder

## Logo Setup (Required!)

To display your logo correctly, place your logo file in this folder:

```
assets/
└── logo.png   ← Put your ProdCast logo PNG here
```

### Steps:
1. Save your logo as `logo.png`
2. Place it in this `assets/` folder
3. The logo should ideally be a **square** file (e.g., 400×400px or 512×512px)
4. PNG format is recommended (supports transparency)

> The website already has this folder set up — just drop your logo.png here and refresh!
