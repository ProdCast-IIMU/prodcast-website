# The Ultimate Product Teardown Framework

A product teardown is not just about clicking through screens. It is a systematic deconstruction of a product's strategy, user psychology, and business viability.

## 1. The Core Loop
Every successful product has a core loop. For Instagram, it's: 
**Create Content → Get Engagement → Feel Rewarded → Create More Content**. 

When doing a teardown, first identify the core loop. If the core loop is broken, the product will fail, regardless of how good the UX is.

## 2. User Psychology & Friction
Identify the moments of high friction. Why did the designer put friction there? 
- **Good Friction:** A confirmation screen before transferring money.
- **Bad Friction:** A 5-step signup process to read a single article.

## 3. Monetization Engine
How does this product make money?
- **SaaS (Subscription):** Predictable recurring revenue.
- **Marketplace (Take Rate):** Taking a cut of every transaction (e.g., Uber, Airbnb).
- **Ads (Attention):** Selling user attention to advertisers (e.g., Meta, Google).

> Always ask: Does the monetization strategy align with the user's best interests?
