# Project Constitution

## 1. Project Context
**Prodcast** is the premier product management club at IIM Udaipur. This website serves as the central hub for our community, showcasing our events, mission, and resources. 

## 2. Design Archetype: Clean Modern Minimalism
- **Aesthetic**: Stark whites, deep blacks, subtle greys. High contrast, Apple/Vercel-inspired minimalism. 
- **Typography**: Perfectly kerned sans-serif (Inter) for brutal legibility, accented by JetBrains Mono for metadata.
- **Motion**: Subtle 2D scroll transitions, graceful fade-ins, and micro-animations. No heavy 3D or deep space environments.
- **Priority**: Flawless readability and user experience above all else.

## 3. Technical Stack
- **Framework**: React via Vite
- **Styling**: Tailwind CSS v3 (Utility-first approach, avoiding inline styles)
- **Animation**: Framer Motion & Lenis (Smooth scrolling)
- **Routing**: React Router DOM

## 4. Core Principles
- **Performance First**: Zero WebGL/Three.js. Small JS bundle.
- **Responsive & Accessible**: Strict adherence to web standards.
- **Spec-Driven Development**: Update the SPECIFICATION.md before implementing major structural changes.
